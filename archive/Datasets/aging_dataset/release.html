AnAge Build 14: Release Notes

Release date: October 14, 2017
Number of entries: 4,244

Build 14 is primarily a qualitative improvement of AnAge. Only a few new species were added, but dozens of entries were updated, ~150 new references were added and the taxonomy was updated. As previously, the release of a new build means that HAGRID numbers were reshuffled.

Please remember that these notes refer to the latest stable build of AnAge. The current AnAge stats may differ slightly from the above as a result of changes after the build's release.
