<p class="read-more-wrap">
<font style="color:teal">
<a href="https://infoscience.epfl.ch/record/226751" target="_blank">New paper</a> accepted in the IEEE conference on <a href="https://www.ieee-security.org/TC/SP2017/" target="_blank">Security and Privacy, 2017</a>.</font>
<span class="read-more-target"><font style="color:gray;font-size:80%">Currently most smartphones allow users to set their privacy preferences only during the installation of an app (e.g. answers to questions like "Facebook would like to access your location: allow/deny?"). Such <i>static</i> policies may force users to overshare sensitive data because it does not take the context into account (e.g. sharing location might be ok when we are using the app, but not ok when the app is running in the background).
In this paper, we propose a machine learning framework to automatically make such decisions on a user's behalf. Our method uses context to achieve an accuracy of around 80% which is a 50% improvement over the static-policies currently used in Android and iOS.</font></span>
</p>

<h4 style="font-weight:bold">Research</h4>
<p><i>"My main goal is to understand the principles of learning from data and use them to develop algorithms that can learn like living beings."</i></p>
<p>My current focus is to understand the role of uncertainty in learning and to develop fast algorithms for uncertainty estimation.</p>

<h4 style="font-weight:bold">Recent Research Highlights</h4>
<ul class="w3-ul">
<li> <a href="https://arxiv.org/abs/1703.04265" target="_blank">Conjugate-Computation Variational Inference (CVI)</a> generalizes Stochastic Variational Inference (SVI) to non-conjugate models.
<li>Applications:
<a href="papers/Building_Simulation.pdf" target="_blank">GPs for building design</a>,
<a href="papers/bayesopt-2015.pdf" target="_blank">Baysian optimization with UAVs</a>,
<a href="https://infoscience.epfl.ch/record/226751" target="_blank">Automatic privacy permission for Smart-phones</a>,
<a href="http://vincent.etter.io/publications/etter2016dsaa.pdf" target="_blank">Analysis of vote results</a>,
and <a href="http://infoscience.epfl.ch/record/196605/files/khan14.pdf" target="_blank">Preference learning</a>.
</ul>

</div>
